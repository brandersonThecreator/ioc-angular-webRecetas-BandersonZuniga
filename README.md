# ReceptaHub - Brander Zunsad

## Descripció
ReceptaHub és una aplicació web que funciona com una biblioteca de receptes de cuina. Els usuaris poden explorar diferents plats, descobrir noves idees per cuinar i consultar receptes de manera senzilla.

A més, la plataforma permet que els usuaris comparteixin les seves pròpies receptes amb la comunitat, creant així un espai col·laboratiu on es poden descobrir noves preparacions i inspirar-se per cuinar.

## Stack Tècnic
- Angular 18
- TypeScript
- SCSS
- Standalone Components
- HTML
